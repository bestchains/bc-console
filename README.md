## 开发构建

### 快速开始

克隆项目文件:

```bash
git clone git@github.com:bestchains/bc-console.git
```

进入目录安装依赖:

```bash
pnpm install
```

开发：

```bash
npm dev
```

构建：

```bash
npm build
```

代码检测：

```bash
npm run lint
```
> 注意事项: lint 规则默认忽略了 js、jsx 及 index.css 文件，这些文件一般都是低代码平台自动生成的，如果要手动开发页面，请使用 ts、tsx 及 less。
